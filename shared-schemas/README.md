# Shared Schemas

Shared JSON Schemas for StoryComicAI domain contracts.

## Included MVP Schemas
- `story-plan.schema.json`
- `page.schema.json`
- `panel.schema.json`
- `dialogue.schema.json`
- `character-profile.schema.json`
- `generation-job.schema.json`
- `analytics-event.schema.json`

These schemas are designed to map cleanly to Pydantic models used by API and worker services.
